Задача 8.(Сложная)

У нас есть таблицы:

products:
	- product_id INT PK
	- product_name TEXT
	
order:
	- order_id INT
	- product_id INT

Каждая строка в order — это товар в заказе. В одном заказе может быть несколько товаров. (Каждый заказ всегда имеет 2 и более товара)

Найти пару товаров, которые чаще всего покупают вместе(встречаются в одном заказе) и вывести их имена.
