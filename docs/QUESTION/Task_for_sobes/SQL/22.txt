Задача 22.(Средняя)

1. Необходимо создать и внести данные в таблицу. Таблица имеет два столбца.
— Первый — уникальный номер (ID) от 1 до 100.
— Второй — Цвет. Если значение ID делится на 3, то цвет КРАСНЫЙ, если ID делится на 4, то цвет СИНИЙ, если ID делится и на 3 и на 4, то цвет ФИОЛЕТОВЫЙ.

2. Выведите кол-во ID каждого цвета. Оставте в выборке только те цвета, в которых кло-во ID > 30.

3. Выведите значение первых 3х ID каждого цвета, которые заканчиваются на цифру 7.

4. Выведите все ID отсортированные в обратном порядке, а так же значения, сколько ID было перед ними с заполненым полем "Цвет".
