Задача 43.(Средняя)

Представим, что у нас есть интернет-магазин, из которого мы собираем данные.

1. Данные по логинам пользователя

Данные летят в нашу таблицу в режиме реального времени, каждый раз, когда пользователь логинится на сайт.

Таблица: user_login
Поля:
user_id UInt64
login_time DateTime
Объём данных большой

2. Данные по акциям

Данные летят в нашу таблицу в режиме реального времени, каждый раз, когда создаётся новая акция.

Таблица: event
Поля:
event_name String
start_time DateTime
end_time DateTime
Объём данных маленький

Задача

Посчитать количество логинов пользователей во время действия акций за выбранный промежуток времени.

Использовать только синтаксис ClickHouse
