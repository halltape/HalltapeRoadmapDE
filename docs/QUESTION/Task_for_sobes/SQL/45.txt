t_agr_cred_optn — таблица операций по кредитному договору:
agr_cred_id text — id договора
optn_id text — id операции
optn_dt date — дата операции
optn_dttm timestamp — дата-время операции
optn_amt numeric — сумма операции
optn_inc_flag boolean — признак: операция увеличивает или уменьшает остаток по счёту

Задача: написать запрос, который вернёт историю изменения суммы exposure по договору с полями:

agr_cred_id
exposure_amt
start_dt
end_dt
