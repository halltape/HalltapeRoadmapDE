Задача 10.(Сложная)

Дана таблица hotel_logs с логами отеля

Поля:

id — id клиента
date_to — дата въезда клиента
date_from — дата выезда клиента

Пример данных:

id  | date_to  |date_from |
1111|2022-01-01|2022-01-10|
1112|2024-02-27|2024-03-10|
1111|2025-06-06|2999-01-01|

Задания:

1. Посчитать, сколько было посетителей в отеле на каждый день.
2. Выделить топ загруженных сезонов.
3. Определить месяцы, когда отель пустует.
