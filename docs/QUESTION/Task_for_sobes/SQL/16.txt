Задача 16.(Простая)

 Дана таблица PLU_GROUP. 
 Содержит историчную информацию товар - группа товара,
 с периодом действия группы товара для каждого товара.

 ---------------------------
 PK  |plu_id         |int
 FK  |group_n        |varchar(20)
 PK  |from_dt        |date
     |to_dt          |date
 ---------------------------
     
 Написать SQL запрос, который за 1 запрос выведет: 
 1. Всю таблицу PLU_GROUP с сортировкой по коду товара, периоду действия записи
 добавить информацию:
 2. Поле `vers_err`  - отметить строки с ошибками версионности 
 3. Поле `dupl_rnbr` -  количество строк по `plu_id` и `group_nm` 
 4. В поле `total_rnbr` - общее количество строк с ошибками

 Периоды должны следовать последовательно,
 начало нового периода from_dt date = to_dt + 1 day
